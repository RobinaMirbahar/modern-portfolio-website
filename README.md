# Professional Portfolio Website

A modern, responsive portfolio website crafted with **Google AI Studio** using the full-stack vibe coding experience. Built with React 19, Vite, Tailwind CSS, and Motion animations—this portfolio showcases professional work, skills, and experience with a clean, modern aesthetic.

**Live Demo:** [View Portfolio](https://[your-username].github.io/[repository-name]/)

## 🤖 Built with AI Studio

This project was created using [Google AI Studio's full-stack vibe coding](https://blog.google/innovation-and-ai/technology/developers-tools/full-stack-vibe-coding-google-ai-studio/) experience—a new way to build production-ready applications from prompts. The AI agent handled:

- Project scaffolding with React + Vite
- Tailwind CSS 4 configuration

- Responsive design implementation
- Dark mode theming

- ⚡ **Lightning Fast** – Built with Vite for optimal performance

## 🚀 Run Locally

### Prerequisites
- Node.js 18+ ([Download](https://nodejs.org))

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/[your-username]/[repository-name].git
   cd [repository-name]
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173`

## 🚢 Deployment

### GitHub Pages (Automatic)
Push to `main` and GitHub Actions will deploy automatically.

### Manual
```bash
npm run build
# Deploy the dist folder to your hosting service
```
